# 🚀 Clarion AI — PWA + Mobile Setup Guide

This version of Clarion AI has been upgraded with:
- ✅ **Progressive Web App (PWA)** — installable on any device
- ✅ **Mobile-Responsive UI** — collapsible sidebar + bottom nav
- ✅ **Capacitor** config — ready for native Android/iOS builds
- ✅ **Voice Input** — microphone support on analyze page
- ✅ **Offline Page** — graceful offline experience
- ✅ **PWA Install Prompt** — smart install banner for users

---

## 📦 Step 1: Install Dependencies

```bash
cd frontend
npm install
```

This installs everything in `package.json`, including:
- `next-pwa` — auto service worker generation
- `@capacitor/core`, `@capacitor/cli` — native app wrapper
- `@capacitor/android`, `@capacitor/ios` — platform support

---

## 🌐 Step 2: Run as PWA (Web)

```bash
cd frontend
npm run dev         # Development (PWA disabled in dev)
npm run build       # Production build (PWA enabled)
npm run start       # Serve production build
```

After `npm run build`, open in Chrome → address bar → install icon appears.

### PWA Features Added:
| Feature | File |
|---------|------|
| Web App Manifest | `public/manifest.json` |
| Service Worker | Auto-generated by `next-pwa` during build |
| Custom SW logic | `public/sw-custom.js` |
| Icons (all sizes) | `public/icons/icon-*.png` |
| Offline page | `src/app/offline/page.js` |
| Install prompt | `src/components/PWAInstallPrompt.js` |
| iOS meta tags | `src/app/layout.js` |

## 📱 Step 3: Mobile UI Features

### What's new:
- **Hamburger menu** in TopNav opens sidebar on mobile
- **Sidebar** slides in with dark overlay on mobile (`lg:` breakpoint = desktop)
- **Bottom Navigation** appears on mobile with 5 quick-access tabs
- **Voice Input** on Analyze page (tap microphone icon)
- **Safe area** support for iPhone notch/Dynamic Island
- **Touch targets** enlarged to 44px minimum

---

## ⚡ Step 4: Build Native App with Capacitor

```bash
cd frontend

# 1. Build web app for static export
BUILD_TARGET=capacitor npm run build

# 2. Add Android platform
npx cap add android

# 3. Sync web build into native project
npx cap copy

# 4. Open in Android Studio
npx cap open android

# For iOS (requires Mac + Xcode):
npx cap add ios
npx cap open ios
```

### Capacitor Config: `capacitor.config.json`
- App ID: `ai.clarion.app`
- Web directory: `out` (static export)
- Status bar: Dark + orange theme
- Keyboard resize support

## 🎤 Voice Input

The Analyze page now has a **Voice** button that uses `Web Speech API`.

- ✅ Works in Chrome (desktop + Android)
- ✅ Works in Safari (iOS 14.5+)
- ❌ Not available in Firefox
- 🔐 Requires microphone permission

Hook location: `src/hooks/useVoiceInput.js`
